# Feeder shape specimens

These files are privacy-safe shape specimens derived from feeder formats observed
during Mission Control development. All identifiers, paths, titles, messages, and
repository names are synthetic; no live transcript content or operator data is
stored here.

The dashboard and chat-graph acceptance suites copy these specimens into isolated
temporary homes and run the real collectors against them. This makes an upstream
shape change fail at the collector boundary instead of silently degrading a live
dashboard.

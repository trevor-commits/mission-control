# Feeder shape specimens

These files are privacy-safe specimens of feeder outputs and provider records
observed during Mission Control development. All identifiers, paths, titles,
messages, and repository names are synthetic; no live transcript content or
operator data is stored here.

The dashboard suite runs its real command-output wrapper against the usage and Git
outputs. The chat-graph suite copies provider records into isolated temporary
stores and runs the real graph collectors and transcript scanner. These are
consumer-boundary drift checks; they do not claim to execute the upstream usage or
repository-state generators from their own outputs.
